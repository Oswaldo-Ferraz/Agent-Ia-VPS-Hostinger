## 📁 NOVA ESTRUTURA

```
🧠 Agente-IA-VPS-Hostinger/
├── 🤖 src/           # Código fonte
├── 📚 docs/          # Documentação
├── 🔧 scripts/       # Scripts de automação
└── ⚙️ config/        # Configurações
```

### **🤖 src/** - Código Fonte
- `agents/` - Agentes principais (IA, VPS, Hostinger)
- `services/` - Serviços auxiliares (wp-manager)
- `utils/` - Utilitários (show-wp-sites)

### **📚 docs/** - Documentação
- `guides/` - Guias detalhados
- `tutorials/` - Tutoriais passo-a-passo
- `examples/` - Exemplos práticos
- `setup/` - Configuração e instalação

### **🔧 scripts/** - Scripts
- `setup/` - Scripts de configuração
- `tests/` - Scripts de teste

### **⚙️ config/** - Configurações
- `examples/` - Templates de configuração
- `templates/` - Templates para geração
